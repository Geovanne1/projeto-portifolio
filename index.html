Sobre os Projetos

Barbearia

O projeto da Barbearia foi uma sequência de cursos sobre html e css.

No html foi ensinado sobre a estrutura básica do documento, as tags, a separação que é feita entre a head(cabeça) e o body(corpo) da página.

Na cabeça fica todas as informações que a página vai usar para funcionar, como por exemplo as meta tags a localização do arquivo css entre outras.

No corpo fica toda a parte que será exibida para o usuário. O corpo é dividido em trés partes header(cabeçalho), main(conteúdo) e o footer(rodapé), e dentro deles tem várias subdivisões que pode ou não serem atribuídas.

Acessibilidade

O projeto de acessibilidade foi pra mostrar a importância de deixar o seu projeto o mais acessível possível, e ensina como fazer isso.

O site foi testado com o software NVDA, um leitor de tela para pessoas com eficiência visual, todas as imagens com uma descrição a estrutura da página totalmente repensada para um melhor intendimento.

Flexbox/Grid

O projeto de Flebox/Grid tem um foco no posicionamento com estilos(css) uma funcionalidade do css que vem facilitando muito para os programadores.

Antes dessas ferramentas o posicionamento de elementos era uma tarefa que exigia muito tempo e tinha muitas complicação, principalmente para responsividade(ajuste da layout para telas de diferentes tamanhos).

Grid

O projeto Grid, é um aprofundamento a ferramenta.

Por se tratar de uma ferramenta com várias funcionalidades diferentes é um ótimo projeto para conhecer um pouco mais sobre cade uma delas

Arquitetura-css

O projeto tem grande parte do seu foco em mostrar as convenções usadas pala comunidade de programadores.

Como nomear as classes, como organizar o css em várias pastas para facilitar caso tenha que ser feita uma mudança no design da página, organizar todas as ferramentas por ondem alfabética para facilitar a localização da cada uma delas

JavaScript-Web

O projeto JavaScript mostra as principais ferramentas que podem ser usadas para implementar funcionalidades que não são possíveis só com html e css.

É possíveis adicionar um paciente, procurar os pacientes pelo nome, o calculo do imc é feito automaticamente, caso tente adicionar um paciente que não tenha nem o peso e nem a altura permitida o sistema não deixa.

O código pega uma lista externa de pacientes e implementa na tabela, caso tenha digitado errado ou queira apagar um paciente da tabela basta clicar duas vezes no paciente que ele será eliminado.

Dinamismo

O projeto de Dinamismo tem como foco mostrar de uma forma interativa a implementação de JavaScript usando botões para ativer elementos ocultos no código.
